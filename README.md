<a name="english"></a>
## English

**CoFlux** is a powerful AI-driven knowledge management system designed for proactive insight generation and semantic networking of your ideas.

### 🚀 Key Features
- **Proactive AI Insights**: Automatically generates related ideas and summaries based on your writing.
- **Semantic Knowledge Map**: Visualize your knowledge as a 2D/3D semantic web using MDS-based force layouts.
- **Knowledge Heatmap**: Track your activity and knowledge density across different topics.
- **Local-First & Privacy**: Built on SQLite for zero-cost persistence and local-first data ownership.
- **AI Workflows**: Custom AI-driven templates and scriptable automation for complex tasks.
- **Cross-Platform**: High-performance desktop experience powered by Tauri and Rust.

### 🛠️ Tech Stack
- **Frontend**: React, Typescript, Vite, Lucide React
- **Backend/Core**: Rust, Tauri, Rusqlite (SQLite)
- **CRDT/Sync**: Yjs for real-time collaborative editing foundations
- **AI/ML**: Integrated with OpenAI, Ollama (Local LLM), and Supabase Vecs

### 📦 Quick Start
1. Install dependencies: `npm install`
2. Run in development: `npm run tauri dev`
3. Build for production: `npm run tauri build`

## 한국어

**CoFlux**는 창의적인 아이디어의 연결과 실시간 통찰력을 제공하는 강력한 AI 기반 지식 관리 시스템입니다.

- **프로액티브 AI 인사이트**: 사용자의 기록을 분석하여 관련 아이디어와 요약을 자동으로 제안합니다.

- **로컬 우선 및 프라이버시**: SQLite 기반의 제로 코스트 영속화로 데이터 소유권을 보장합니다.
- **AI 워크플로우**: 사용자 정의 AI 템플릿과 스크립트 기반 자동화를 지원합니다.
- **크로스 플랫폼**: Tauri와 Rust를 사용한 고성능 데스크톱 앱 환경을 제공합니다.

- **AI/ML**: OpenAI, Ollama (로컬 LLM), Supabase 연동 지원

## Русский

**CoFlux** — это мощная система управления знаниями на базе ИИ, предназначенная для проактивной генерации идей и создания семантических связей.

### 🚀 Ключевые особенности
- **Проактивные ИИ-инсайты**: Автоматическая генерация связанных идей и резюме на основе ваших записей.
- **Семантическая карта знаний**: Визуализация связей между вашими идеями с помощью MDS-раскладки.
- **Тепловая карта знаний**: Отслеживание активности и плотности знаний по различным темам.
- **Локальное хранение и приватность**: Нулевая стоимость персистентности на базе SQLite и полный контроль над данными.
- **ИИ-воркфлоу**: Настраиваемые шаблоны и автоматизация на базе ИИ.
